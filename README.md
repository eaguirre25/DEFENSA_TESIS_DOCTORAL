# Defensa de tesis doctoral

Presentación interactiva de la defensa doctoral de Elías Aguirre.

## Ver la presentación

[Abrir la presentación](https://eaguirre25.github.io/DEFENSA_TESIS_DOCTORAL/)

No requiere instalación. Se recomienda utilizar una computadora, un navegador actualizado y pantalla completa.

## Controles

- Flechas: avanzar o retroceder.
- `0`: salir de una pantalla embebida.
- `F`: activar o desactivar pantalla completa.
